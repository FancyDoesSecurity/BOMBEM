#BOMBEM
## What's BOMBEM?
```
bombem is a tool meant for windows post exploitation, it aids in the process of:
    * Privilege Escalation
    * Credentials Harvesting
    * 'Looting' (Extraction Of Files, Etc)
    * System & Network Enumeration
    * Persistence
```
## Usage
```
git clone https://github.com/FancyDoesSecurity/BOMBEM.git
cd BOMBEM
windowsInstall.ps1
python3 windows-bombem.py
```
